Objective

Given the root of a binary tree, invert the tree, and return its root. Every node's left and right children must be swapped.

Example 1

Input: root = [4,2,7,1,3,6,9]
Output: [4,7,2,9,6,3,1]

Constraints

  • The number of nodes in the tree is in the range [0, 100].
  • -100 <= Node.val <= 100
info

Tip: Consider using recursion to traverse down the tree.

class Solution:
def invertTree(self, root: Optional[TreeNode]) -> Optional[TreeNode]:
# BASE CASE: If the node is null, return None
if not root:
return None
 
# SWAP: Exchange left and right children
root.left, root.right = root.right, root.left
 
# RECURSION: Attack the children nodes
self.invertTree(root.left)
self.invertTree(root.right)
 
return root
